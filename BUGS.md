# BUGS

## Баги на скриншоте (Задание 1)

### BUG-SCR-001 - На странице "Котята в Москве" присутствует объявление из другого города
- **Краткое описание:** в выдаче виден товар с локацией `Новосибирск`, хотя заголовок страницы - "Котята в Москве".
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания.
- **Фактический результат:** в карточках есть локация `Новосибирск`.
- **Ожидаемый результат:** в выдаче по Москве должны показываться объявления только из Москвы (или должна быть явно обозначена логика смешанной выдачи).
- **Серьёзность / приоритет:** Major / **P1**
- **Окружение:** скриншот из задания `bags.png`.

### BUG-SCR-002 - В интерфейсе отображается сообщение об ошибке на рабочей странице
- **Краткое описание:** в правой верхней части контента виден текст "Ошибка, обновите страницу".
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания.
- **Фактический результат:** сообщение об ошибке отображается одновременно с основной выдачей.
- **Ожидаемый результат:** при исправной работе страницы не должно быть ошибок в UI.
- **Серьёзность / приоритет:** Major / **P1**
- **Окружение:** скриншот из задания `bags.png`.

### BUG-SCR-003 - Количество найденных объявлений не совпадает с количеством в кнопке применения фильтра
- **Краткое описание:** в заголовке указано `Котята в Москве 19`, но кнопка фильтра показывает `Показать 8 объявлений`.
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания.
- **Фактический результат:** на странице одновременно отображаются взаимоисключающие значения количества.
- **Ожидаемый результат:** количество в заголовке и в CTA-кнопке должно быть согласованным для одного и того же набора фильтров.
- **Серьёзность / приоритет:** Major / **P1**
- **Окружение:** скриншот из задания `bags.png`.

### BUG-SCR-004 - В выдаче присутствует товар, не соответствующий выбранной категории "Животные"
- **Краткое описание:** среди карточек есть "Персидский котёнок плюшевый" (игрушка), что не соответствует категории живых животных.
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания.
- **Фактический результат:** в выдаче категории "Животные -> Кошки" присутствует объявление с игрушкой.
- **Ожидаемый результат:** в категории животных должны отображаться только релевантные живые животные.
- **Серьёзность / приоритет:** Major / **P1**
- **Окружение:** скриншот из задания `bags.png`.

### BUG-SCR-005 - Нарушение релевантности по фильтру породы
- **Краткое описание:** выбран фильтр породы `Персидская`, но в списке есть карточка `Экзотический короткошёрстный котёнок`.
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания.
- **Фактический результат:** отображаются объявления с другой породой.
- **Ожидаемый результат:** выдача должна соответствовать выбранной породе либо иметь явное указание на расширенный поиск.
- **Серьёзность / приоритет:** Major / **P1**
- **Окружение:** скриншот из задания `bags.png`.

### BUG-SCR-006 - Некорректный tooltip на иконке избранного
- **Краткое описание:** у иконки избранного отображается tooltip "Не показывайте мне похожие объявления", который не соответствует действию и выглядит обрезанным.
- **Шаги воспроизведения:** открыть предоставленный скриншот из задания (область карточек объявлений).
- **Фактический результат:** несоответствие текста tooltip действию (избранное) + визуальный дефект текста.
- **Ожидаемый результат:** у иконки избранного должен быть корректный и читаемый tooltip, например "Добавить в избранное".
- **Серьёзность / приоритет:** Minor / **P2**
- **Окружение:** скриншот из задания `bags.png`.
